# TROUBLESHOOTING

#   pip install --upgrade setuptools wheel
#   pip install pywin32
#   python Scripts/pywin32_postinstall.py -install
#
# If JAX is slow:
#   Ensure you're using JIT: @jit decorator on functions
#   First call compiles (slow), subsequent calls are fast

#   Run Python as Administrator
#   Register COM: regsvr32 "C:\Program Files\Gamry Instruments\Framework\GamryCOM.dll"
